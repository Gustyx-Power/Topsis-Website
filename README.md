# Topsis-Website

A website for comparing smartphone specifications using the TOPSIS method.

## Project Structure

This project consists of several HTML, CSS, and JavaScript files that work together to provide the smartphone comparison functionality.

### HTML Files

-   `index.html`: The main entry point of the website. It likely contains the primary interface for users to start the comparison process.
-   `ranking.html`: This page displays the final ranking of the smartphones based on the TOPSIS calculation.
-   `comparison.html`: This page shows a side-by-side comparison of the specifications of selected smartphones.

### CSS Files

-   `css/styles.css`: Contains all the styles for the website to ensure a consistent look and feel across different pages.

### JavaScript Files

The JavaScript files are located in the `js/` directory and are responsible for the dynamic functionality of the website.

-   `js/app.js`: The main application script that handles the overall application logic.
-   `js/data.js`: This file holds the raw data for the smartphones, such as their specifications and attributes.
-   `js/comparison.js`: Contains the logic specific to the `comparison.html` page.
-   `js/components.js`: Contains reusable UI components or functions.
-   `js/utils.js`: A set of utility functions used across different parts of the application.

## How to Use

1.  Open the `index.html` file in your web browser.
2.  Follow the on-screen instructions to select criteria and smartphones.
3.  View the results on the `ranking.html` and `comparison.html` pages.
